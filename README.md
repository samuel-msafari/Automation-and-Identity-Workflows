**M365-Inactive-User-Audit.ps1** This script identifies inactive users in Microsoft Entra (Azure AD) by checking their sign-in activity over the last 90 days and exports the results to a CSV file.
**M365-Automated-Offboarding.ps1** This script disables an offboarded user's Entra ID account and converts the mailbox to a shared mailbox to preserve mailbox data.
